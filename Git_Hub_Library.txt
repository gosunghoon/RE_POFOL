연결 : git remote add <remote-name> <git address>
연결 확인 : git remote --verbose
연결 삭제 : git remote remove <remote-name>

올릴 파일 기록하기  : git add <filename>
한번에 올리기 : git add . 
특정 폴더내부만 올리기 : git add /new/*
특정 문서만 올리기 git add *.txt

수정사항만 올리기 : git add --patch <filename>.txt


올라간 파일 Commit하기 : git commit -m "파일이름.txt"
깃허브로 올리기 : git push <깃이름> main

강제 전송 : git push <폴더이름> main --force
git push RE_POFOL master --force


**main이 master일수도 있다.

저장소 연결 후 제대로 연결되었는지 확인한 다음
올릴파일의 폴더로 cd해서 이동 후 기록하기
그리고 올라간 파일을 Commit과 동시에 push해주기



브런치 삭제 : git branch -d 브런치이름
현재 브런치 명 변경 : git branch -m 이름
브런치 목록 확인 : git branch
